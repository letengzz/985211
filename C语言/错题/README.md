##### 2. 设x为int类型，其值为11，则表达式（x++*1/3）的值是（ ）

  `A. 3`   `B. 4`   `C. 11`   `D. 12`

解析： x先乘以1,然后再除以3,得结果3!
计算完后x自加为12!****
