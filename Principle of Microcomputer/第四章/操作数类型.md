# 操作数类型

## 立即数操作数

​	表示参加操作的数据本身，可以是8位或16位

  **例**： 

```
	MOV AX，1234H    ;AX--》1234H

	MOV BL，22H 
```

- **立即数无法作为目标操作数**

- **立即数可以是无符号或带符号数，其数值应在可取值范围内**

## 寄存器操作数

​	表示参加运算的数存放在指令给出的寄存器中，可以是16位或8位。

 **例**：

```
	MOV AX，BX 

	MOV DL，CH
```

## 存储器操作数

​	表示当前参加运算的数存放在存储器的某一个或某两个单元中

**例**：

```
	MOV AX，[1200H]

	MOV AL，[1200H]
```
