# 寻址方式

​	计算机可以对存储器、寄存器中的数据进行各种操作，如取数、存数、加运算、减运算等，这就需要找到数据的存放地址，再对其中的数据执行相应的操作，最后将运算结果送往指定的地址单元或寄存器。**指令的寻址方式就是寻找指令操作数所在地址的方式**，以确定数据的来源和去处。熟练地掌握寻址方式对学习指令系统和汇编语言程序设计具有重要作用。
