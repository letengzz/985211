# 指令

**指令**：指令是指示计算机完成特定操作的命令

**指令系统**：指令系统是计算机能够执行全部命令的集合，它取决于计算机的硬件设计。指令系统因机而异，没有通用性(可移植性)。

![1](D:/Data/typora/photo/1.png)

**指令格式**：

```
操作码 [操作数1],[操作数2]
```

- **操作码**：执行何种操作
- 操作数1：**目的操作数**(指令加工之后形成的数据)
- 操作数2：**源操作数**(指令加工之前的数据)

[参加操作的数据]()：**目的操作数**、**源操作数**
