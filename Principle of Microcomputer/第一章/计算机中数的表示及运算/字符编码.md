# 字符编码

**(ASCII码 American Standard Code For Information Interchange，美国标准信息交换码)** 

一般用来**显示** 或**传输**  用

可表示**128种字符的7位基本ASCⅡ码**和**可表示256种字符的8位扩充ASCⅡ码**（可重新定义）。
  字符可分为：**显示字符**和**控制字符**。
    0—9：ASCII码	`30H`—`39H`
    A—Z：                 `41H`—`5AH`
    a—z：                  `61H`—`7AH`
