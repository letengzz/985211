# DRAM芯片4116

存储容量为16K × 1

十六个引脚：

- A<sub>0</sub>~A<sub>6</sub>：地址线
- <SPAN style="TEXT-DECORATION: overline">WE</SPAN>：读/写控制线，WE=1为读出。WE=0为写入
- <SPAN style="TEXT-DECORATION: overline">RAS</SPAN>：行选通信号
- <SPAN style="TEXT-DECORATION: overline">CAS</SPAN>：列选通信号
- D<sub>in</sub>：数据输入
- D<sub>out</sub>：数据输出
